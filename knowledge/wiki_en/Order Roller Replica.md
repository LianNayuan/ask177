# Order Roller Replica

The Order Roller Replica is a main weapon in Splatoon 3 .

## Splatoon 3

- **Class**: Roller
- **Sub**: Curling Bomb
- **Special**: Big Bubbler
- **Special points**: 180p
- **Range**: 48/100
- **Ink speed**: 45/100
- **Handling**: 55/100
- **Level**: 0
- **Added in**: Side Order ( 7.0.0 )
- **Base damage**: 35–150 (Ground) 40–150 (Midair) 125 (Roll)
- **Other variants**: Splat Roller Krak-On Splat Roller

## Appearance

The Order Roller Replica is designed to look like a simplified version of the Splat Roller , containing only white and gray colors.
