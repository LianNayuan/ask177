# Order Splatana Replica

The Order Splatana Replica is a main weapon in Splatoon 3 . It is a reskin of the Splatana Stamper .

## Splatoon 3

- **Class**: Splatana
- **Sub**: Burst Bomb
- **Special**: Zipcaster
- **Special points**: 210p
- **Range**: 75/100
- **Damage**: 43/100
- **Handling**: 60/100
- **Level**: 0
- **Added in**: Side Order ( 7.0.0 )
- **Base damage**: 35 (Projectile-tap) 20 (Direct-tap) 70 (Projectile-charge) 140 (Direct-charge)
- **Ink consumption**: 5.2% (tap) 11.7% (hold)
- **Other variants**: Splatana Stamper Splatana Stamper Nouveau Stickerz Splatana Stamper

## Appearance

The Order Splatana Replica is designed to look like a simplified version of the Splatana Stamper , containing only white and gray colors.

The Order Splatana Replica is the only splatana with a Stamper model that does not feature a rotating stamp track when charging the weapon.
