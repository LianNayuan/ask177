# 武器俗称及来源

| 官中译名 | 俗称 | 俗称由来 |
|---|---|---|
| 广域标记枪 | 喇叭 | 外形酷似喇叭，故称为喇叭。日文翻译成英文为 Bold Marker，即粗马克笔，和第三个针管是兄弟关系。 |
| 新叶射击枪 | 新叶 | 日文翻译而来，官中一致。 |
| 窄域标记枪 | 针管 | 枪口部分为针管笔，故简称为针管。另外，日文翻译成英文为 Sharp Marker，即尖马克笔。 |
| 专业模型枪 MG | 银喷 | 原型是涂装用的喷枪，结合颜色故称为银喷。官方译名为日文直译，据群友介绍，MG 是高达模型系列之一的缩写。 |
| 斯普拉射击枪 | 小绿 | 一代的 Splattershot 非常绿，故被称为小绿。叫法一直沿袭，哪怕二代后续出了黑色贴牌版本，叫法仍为黑小绿，可见叫法的深入人心。 |
| .52 加仑 | 52 | 略 |
| N-ZAP85 | 85 | 略 |
| 顶尖射击枪 | 精英枪 | 日文翻译为英文为 Prime Shooter，官中和通俗叫法的意思都比较接近就是了。 |
| .96 加仑 | 96 | 略 |
| 喷射清洁枪 | 蓝管 | 名字起源有点难以追溯，姑且认为是长得像管子，搭配颜色，有知道的欢迎补充。 |
| 新星爆破枪 | 白泡 / Nova | 白色 + 泡类武器，故称为白泡，Nova 也就是新星的意思，日语和官中都一样。注：爆破枪被称为泡 |
| 火热爆破枪 | 热泡 | 日文名翻成英文为 hot blaster，hot + 泡类武器，故称为热泡。 |
| 远距爆破枪 | 长热泡 | 是热泡，但是更长，故称为长热泡。 |
| 冲涂爆破枪 | 蜡笔泡 | 枪的身上是一圈蜡笔 + 泡类武器，故称为蜡笔泡。 |
| 快速爆破枪 | 红泡 | 红色 + 泡类武器，故称为红泡。 |
| 快速爆破枪 精英 | 长红泡 | 红泡，但是更长，故称为长红泡。 |
| L3 卷管枪 | L3 | 略 |
| H3 卷管枪 | H3 | 略 |
| 开瓶喷泉枪 | 香槟 | 武器原型是香槟，故称为香槟。 |
| 碳纤维滚筒 | 碳刷 | 日语中为 Carbon Roller，即碳 + 刷。注：滚筒被称为刷子 |
| 斯普拉滚筒 | 中刷 | 中型武器 + 刷，故称为中刷。注：轻、中、重为武器重量等级 |
| 电动马达滚筒 | 重刷 | 重型武器 + 刷，故称为重刷。 |
| 可变式滚筒 | 钢笔（刷） | 原型的一部分是钢笔，故称为钢笔（刷）。 |
| 巴勃罗 | 笔刷 / 小毛笔 | 原型就是笔刷，官中与日文一致，该叫法可能是取自画家巴勃罗·毕加索。 |
| 北斋 | 北斋 | 略 |
| 鱿快洁 α | 洗洁精 | 原型是清洁剂，官中译名是日文的直译，似乎和日本的清洁剂品牌有关。 |
| 斯普拉蓄力狙击枪 | 绿狙 | 二代绿色 + 狙，因为是中型武器，也有中狙的叫法。 |
| 斯普拉准心枪 | 镜绿狙 | 绿狙的带镜版本，故称为镜绿狙，也有镜绿 / 镜狙的简称。 |
| 公升 4K | 4K | 略 |
| 4K 准心枪 | 镜 4K | 4K 的带镜版本，故称为镜 4K。 |
| 14 式竹筒枪·甲 | 竹子 / 竹狙 | 原型一部分是竹子 + 狙，故称为竹子 / 竹狙。 |
| 高压油管枪 | 水管 | 原型和日文原名似乎就是运油的管子，大概是因为这武器太水，所以变成了水管（后半句我编的）。 |
| 飞溅泼桶 | 红桶 | 二代红色 + 桶，就是不知道日后是会变成紫桶，还是会和小绿一样，尽管颜色更迭，叫法依旧不变。 |
| 洗笔桶 | 绿桶 | 原型就是笔洗桶，但二代以颜色命名，后续叫法可能更迭，到底是蓝桶还是绿桶呢？ |
| 回旋泼桶 | 洗衣机 | 原型是洗衣机，故称为洗衣机。 |
| 满溢泡澡泼桶 | 浴缸 | 原型是浴缸，故称为浴缸。 |
| 爆炸泼桶 | 重桶 | 重型武器 + 桶，故称为重桶，似乎也有爆炸桶的叫法。 |
| 斯普拉旋转枪 | 轻加 | 轻型武器 + 加特林，故称为轻加。注：旋转枪被称为加特林 |
| 桶装旋转枪 | 中加 | 中型武器 + 加特林，故称为中加。 |
| 消防栓旋转枪 | 消防栓 | 略 |
| 圆珠笔 | 圆珠笔 | 略 |
| 鹦鹉螺弓 47 | 鹦鹉螺 | 略 |
| 溅镀枪 | 红牙（刷） | 原型的一部分是牙刷，因为还有其他颜色的牙刷，故称以颜色区分，称为红牙刷，简称红牙。 |
| 斯普拉机动枪 | 绿双 / 普双 | 绿色 + 双枪，故称为绿双。另外是最普通的双枪绿色 + 双枪，故称为绿双。 |
| 开尔文 525 | 525 / 开尔文 | 略 |
| 双重清洁枪 | 红双 | 红色 + 双枪，故称为红双。 |
| 四重弹跳手枪 黑 | 气垫 / 四滚 | 原型有一部分是气垫，另外武器可以翻滚四次故也有四滚的叫法。 |
| 遮阳防空伞 | 中伞 | 中型武器 + 伞，故称为中伞。 |
| 露营防空伞 | 重伞 / 露营伞 | 重型武器 + 伞，故称为重伞。 |
| 特务配件 | 间谍伞 | 日文翻译成英文为 Splat Gadget，Spat 有间谍的意思，故被称为间谍伞。 |
| 三发猎鱼弓 | 弓 | 略 |
| 工作刮水刀 | 电锯 | 长得有点像电锯，目前不少人称之为电锯。 |
| 雨刷刮水刀 | 刮水刀 | 略 |
