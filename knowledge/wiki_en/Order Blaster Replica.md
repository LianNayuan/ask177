# Order Blaster Replica

The Order Blaster Replica is a main weapon in Splatoon 3 .

## Splatoon 3

- **Class**: Blaster
- **Sub**: Splat Bomb
- **Special**: Zipcaster
- **Special points**: 170p
- **Range**: 18/100
- **Impact**: 70/100
- **Fire rate**: 30/100
- **Level**: 0
- **Added in**: Side Order ( 7.0.0 )
- **Base damage**: 25–35 (Splash) 50–70 (Blast) 125 (Direct hit) 50-70
- **Other variants**: Luna Blaster Luna Blaster Neo

## Appearance

The Order Blaster Replica is designed to look like a simplified version of the Luna Blaster , containing only white and gray colors.
