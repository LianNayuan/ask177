# Order Stringer Replica

The Order Stringer Replica is a main weapon in Splatoon 3 . It is a reskin of the Tri-Stringer .

## Splatoon 3

- **Class**: Stringer
- **Sub**: Toxic Mist
- **Special**: Killer Wail 5.1
- **Special points**: 190p
- **Range**: 88/100
- **Charge speed**: 40/100
- **Mobility**: 40/100
- **Level**: 0
- **Added in**: Side Order ( 7.0.0 )
- **Base damage**: 30–35(x3) (Partial charge) 35(x3) (Full charge) 30 (Projectile explosion)
- **Other variants**: Tri-Stringer Inkline Tri-Stringer Bulbz Tri-Stringer

## Appearance

The Order Stringer Replica is designed to look like a simplified version of the Tri-Stringer , containing only white and gray colors.
